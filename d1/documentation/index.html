Introduction

DefOne Admin Template is a premium admin dashboard template with flat design concept merging in a classic style. It is fully responsive admin dashboard template built with Bootstrap 3+ Framework, HTML5 and CSS3, Media query. It has a huge collection of reusable UI components and integrated with latest jQuery plugins. It can be used for all type of web applications like custom admin panel, project management system, admin dashboard, application backend, CMS or CRM.

We are releasing continuous long term updates and many new features will be coming soon in the near future updates. Once you purchased DefOne, all future updates are free of charge.

LESS

Less is a CSS pre-processor, meaning that it extends the CSS language, adding features that allow variables, mixins, functions. DefOne supports Less and is pure built on top of Less

Less Files are found under d1-v1.4.0/assets/less respectivly d1-v1.4.0/assets/less/modules

            ...
        </nav>


        <nav id="menu" class="navbar-default navbar-fixed-side hidden-xs" role="navigation">
            ...
        </nav>

        
        <div id="page-wrapper" class="fixed-navbar ">
            ...
        </div>

            ...
        </footer>

Here is the list of markup descriptions you will need

  1. The <div id="wrapper">...</div> is mandatory for any page
  2. The <nav>...</nav> is the top navigation container in wich you will need to insert the default Bootstrap Navbar which is explained in the Navbar Section
  3. The second <nav>...</nav> container is the sidebar which is explained in the Sidebar Section of this document.
  4. In the <div id="page-wrapper">...</div> container is the main page where the content goes if you decide to use the fixed Navbar you have to add the class fixed-navbar to the page-wrapper
  5. The <div class="sidebar-right">...</div> where the Chatbar goes

Navbar

The TOP navbar stand by 2 types fixed and static. For the fixed one you'll need to add a class navbar-fixed-top and for the static one navbar-static-top by default is a fixed navbar

<nav class="navbar navbar-default" role="navigation">
    ...
</nav>

Mega Menu

If you want to add a mega menu to the navbar add it in the:

<ul class="nav navbar-top-links navbar-left mega-menu hidden-xs hidden-sm">...</ul>

notice the mega-menu class above? we need it to attach a mega menu to the navbar, let's continue...

<li class="dropdown">

        ...
    </a>
    <div class="dropdown-menu mega-menu">
        ....
    </div>
</li>

and that was how we attach a mega menu to the navbar.

API Reference

Auto-initialized jQuery Plugins

Plugin Implementation example
Bootstrap Tooltips

Add attribute [data-toggle="tooltip"] to any element.

<a href="#" data-toggle="tooltip" data-placement="bottom" title="Print">
    <i class="fa fa-print"></i>
</a>
Bootstrap Popovers

Add attribute [data-toggle="popover"] to any element.

<a href="#" data-toggle="popover" data-placement="bottom" 
    data-content="And here's some amazing content. 
    It's very engaging. Right?" 
    title="Print">
    <i class="fa fa-print"></i>
</a>
Select2

Add attribute [data-toggle="select2"] to any select element.

NEW To change the background image and style of the page heading add attributes:

[data-background="/path/to/background/image"] to add a background image to it

[data-text-color="#FFFFFF"] to change the text color on the page-heading (if the background is to dark and the default text color is not visible enough)

[data-text-shadow] to add a shadow to the text

Helpers

Below we explain how the Helper Classes work wich you can find here: /assets/less/helpers.less

Margin

FIRST PART SECOND PART LAST PART
m for margin t for top; b for bottom; l for left; r for right 0, 5, 10,...,90
Example:

m-t-15 wich means Margin Top 15px
m-l-40 wich means Margin Left 40px

Rule:

Value can not be anything, 1,2,4,5,...,9. It has a step of 5px, eg: 5,10,15,...,90

Other options

To remove padding from a HTML element add the class no-margin or use the m-(l|r|t|b)-0

Padding

FIRST PART SECOND PART LAST PART
p for padding t for top; b for bottom; l for left; r for right 0, 5, 10,...,90
Example:

p-b-10 wich means Padding Bottom 10px
p-r-90 wich means Padding Right 90px

Rule:

Value can not be anything, 1,2,4,5,...,9. It has a step of 5px, eg: 5,10,15,...,90

Other options

To remove padding from a HTML element add the class no-padding or use the p-(l|r|t|b)-0

Misc. Classes

CLASSNAME DESCRIPTION
text-uppercase Sets text-transform: uppercase to the element
text-half-size Set font-size: 10px to the element
no-border Removes border of element
bg-* Change background color of element white|gray|primary|info|warning|success|danger
